Parameters are encoded in smart contracts. The rule version applied to your challenge is recorded on-chain at purchase and cannot change mid-evaluation.
</Info>

| Profit target | 10% |
| Minimum trading days | 5 |

## LP-side configurables

Set by the manager at vault creation.

| Parameter | Default | Manager can set up to |
|---|---|---|
| Min LP deposit | $100 | $50,000 |
| Lockup period | None | 30 days |
| Cooldown before withdrawal | Optional 7 days | Configurable |
| Weekly redemption cap | 20% of AUM | Configurable |
| Early-exit penalty (during lockup) | 2% to insurance fund | 0% = no early exit |
| Withdrawal delay (post-lockup) | ~48h timelock (1h on devnet) | — |

## Risk & access controls

| Parameter | Value |
|---|---|
