Hyro Protocol is a small core program that orchestrates two pluggable layers over cross-program invocations (CPI): **authorization** (policy programs) and **fee calculation** (fee calculators). Neither is hard-coded — both are swappable per vault. That makes the protocol an extensible base to build on.

## The stack

| Layer | Technology |
|---|---|
| Blockchain | Solana |
| Smart contract framework | Anchor 0.31.1 |
| Client SDK generation | Codama (from Anchor IDLs) |
| Oracle transport | NATS with JetStream |
| Off-chain venue integration | CCXT |
| Off-chain engine | Rust |
| Settlement currency | USDC (SPL) |
